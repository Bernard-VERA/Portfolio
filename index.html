Booki

Projet

Création de la page d'accueil d'une agence de voyage


Description

Projet OpenClassrooms (Novembre 2023) :
Vous débutez une alternance en tant que développeur web au sein de la start-up Booki. L’entreprise souhaite développer un site Internet qui permette aux usagers de trouver des hébergements et des activités dans la ville de leur choix. Vous êtes chargé d'intégrer l'interface du site avec du code HTML et CSS. Cette interface devra être responsive et respecter la maquette fournie sur Figma.


Les problématiques

Les problématiques rencontrées sont:
- une mise en page complexe en raison de nombreux éléments, de tailles et formes différentes.
- Des difficultés pour gérer l'affichage lors du passage au format tablette, et mobile.
J'ai pu résoudre ces problèmes en utilisant Flexbox et le Responsive Design.


Compétences

Cette expérience m'a permis d'acquérir des compétences solides en intégration web, notamment dans la manipulation de code HTML et CSS, pour concevoir des interfaces utilisateur intuitives et esthétiques. Je sais maintenant comment intégrer une maquette, en partant de zéro. J'ai aussi découvert comment fonctionnent les Media Queries pour permettre à un site de s'afficher sur tout type d'écran. J'ai aussi appris à utiliser Git et GitHub pour versionner, sauvegarder et partager mon code.

Sophie Bluel

Projet

Créer une page web dynamique avec JavaScript, pour le portfolio de Sophie Bluel.


Description

Projet OpenClassrooms (Décembre 2023):
Vous travaillez comme développeur front-end pour l’agence ArchiWebos qui comprend 50 salariés. Après la réunion pour échanger sur le projet, la cheffe de projet vous envoie en pièces jointes tout ce dont vous aurez besoin pour pouvoir réaliser vos missions, soit: le Figma du design qui vous servira à créer les nouvelles fonctionnalités, le code back-end qui vous permettra de faire persister les données et de tester les fonctionnalités, le code front-end d’origine avec la version statique de la page de l’architecte, et finalement, le Kanban pour la mission pour que vous puissiez voir les tâches qui vous sont attribuées. Vos 3 missions principales sont donc: développer la page de présentation des travaux de l'architecte (à partir du HTML fourni), développer la page de connexion de l'administrateur du site (le client) (code à créer de zéro), et développer la modale permettant d'uploader de nouveaux médias (code à créer from scratch).


Les problématiques

Les problématiques rencontrées sont:
- La récupération des données en passant par une API
- La gestion des évènements et la manipulation du DOM
- La création et la gestion de la modale. Tout cela en JavaScript.
- J'ai aussi dû construire une page de page de connexion pour l'administrateur, avec création et stockage d'un Token d'authentification.
j'ai pu résoudre tous ces problèmes grâce aux cours dispensés par Openclassrooms, mais aussi à mes recherches personnelles sur Internet.


Compétences

Cette expérience m'a permis d'acquérir des compétences solides en Javascript. J'ai appris à récupérer les données utilisateurs dans le JavaScript, via des formulaires et une API. Je sais gérer les évènements utilisateurs et manipuler les éléments du DOM. J'ai aussi découvert le fonctionnement d'un Kanban. J'ai trouvé ce projet trés difficile, mais mon envie d'apprendre a finalement pris le dessus. J'ai maintenant de bonnes bases en Javascript.

Menu Maker by Qwenta

Projet

Planifiez le développement du site de votre client


Description

Projet OpenClassrooms (Janvier 2024):
Vous êtes développeur front-end au sein de l’agence Webgencia. Votre client Qwenta souhaite faire développer un “Menu Maker” : un site permettant aux restaurateurs d’afficher et de mettre en page leurs menus facilement, en quelques clics. Vous disposez de la maquette Figma du site, et des spécifications fonctionnelles. Vous devrez réaliser deux documents:
- Le document des spécifications techniques de Menu Maker
- Un tableau Kanban finalisé, organisant le projet sur toute sa durée.
Ce nouveau projet nécessite beaucoup de recherches pour s’assurer d'un travail de bonne qualité, et aligné sur les dernières technologies ! Vous devrez donc, en parallèle de votre travail, développer un système de veille.


Les problématiques

Les problématiques rencontrées sont:
- La mise en place un système de veille et l'utilisation d'outils spécialisés pour cela.
- Savoir identifier les challenges techniques du projet afin de créer un listing des différents plugins et librairies nécessaires à la création du projet.
- Créer et utiliser un Kanban.


Compétences

Cette expérience m'a permis d'apprendre à rédiger les spécifications techniques d'un projet, à partir de besoins fonctionnels, et de présenter une solution technique. J'ai appris à suivre le déroulement d'un projet grâce à un outil de gestion (Trello), et à découper une fonctionnalité en tâches pour préparer le développement. j'ai aussi mis en place une méthode de veille technologique, pour pouvoir sélectionner les meilleures technologies pour notre projet. Cela m'a permis de voir toutes les facettes de la création d'une application, de l'idée de départ, jusqu'a la mise en production.

Nina Carducci

Projet

Débuggez et optimisez un site de photographe (Nina Carducci).


Description

Projet OpenClassrooms (Janvier 2024):
Vous êtes développeur freelance et vous décidez de proposer vos services d’optimisation SEO à de nouveaux clients. Parmi vos prospects, il y a le site de Nina Carducci, une photographe.


Les problématiques

Les problématiques rencontrées sont:
- Le site était beaucoup trop long lors du chargement (Plus de 7 secondes).
- Structure du code HTML inexistante.
- Problèmes d'accessibilité du site.
- Visibilité du site sur les moteurs de recherche inexistante.
- Nombreux bugs du site.
Les cours d'OpenClassrooms sur le SEO, le référencement local et les technologies d'assistance, ainsi que les outils Lighthouse, Wave et Chrome DevTools, m'ont permis de résoudre ces problèmes assez facilement.


Compétences

Les rapports Lighthouse et les audits Wave étaient catastrophiques. Afin d'optimiser le site, j'ai entrepris des actions telles que la réduction de la taille et du poids des images, l'ajout de balises et d'attributs manquants, l'optimisation de la structure du code, l'ajustement du contraste et la réorganisation des balises. J'ai acquis des compétences en matière de référencement web (SEO), en mettant en œuvre une stratégie de référencement local, en utilisant Schema.org, les balises meta et Google Rich Snippets pour améliorer la visibilité du site sur les moteurs de recherche. J'ai développé des compétences en matière de débogage de sites web, grâce aux Chrome DevTools. J'ai également rédigé un cahier de recette pour tester le site, garantissant son bon fonctionnement et sa fluidité. A la fin du projet, les rapports Lighthouse et les audits Wave étaient plus que satisfaisants.

Kasa

Projet

Créez une application web de location immobilière avec React.


Description

Projet OpenClassrooms (Février 2024):
Vous êtes développeur front-end en freelance pour Kasa, une entreprise de location d’appartements entre particuliers. Kasa fait partie des leaders de la location d’appartements entre particuliers en France. Le site de Kasa a été codé il y a maintenant plus de 10 ans en ASP.NET avec un code legacy important. La CTO, a donc lancé une refonte totale pour passer à une stack complète en JavaScript avec NodeJS côté back-end, et React côté front-end. Votre mission: Démarrer le projet React et développer l’ensemble de l’application, les composants React et les routes React Router, en suivant les maquettes Figma (responsives !). Le recrutement de la personne en charge du back-end n'est pas terminé et va prendre plus de temps que prévu. Du coup, il va falloir faire sans pour le moment. Un fichier JSON contenant les 20 dernières annonces de logements vous permettra de construire le Front qui correspond.


Les problématiques

Les problématiques rencontrées sont:
- La création de la logique de routage.
- La création des animations des menus collapses.
- La création d'une page 404 pour gérer les erreurs.
- La création d'un carrousel d'images.
- Les effets de style, avec SASS
j'ai pu résoudre tous ces problèmes grâce aux cours dispensés par Openclassrooms, mais aussi à mes recherches personnelles sur Internet.


Compétences

J'ai appris à initialiser une application avec Create React App. Avec React Router, je sais maintenant configurer la navigation entre les pages d'une application. L'utilisation de composants React pour développer l'interface du site, m'a appris à structurer efficacement le code, à le rendre modulaire et réutilisable. Cette compétence s'est avérée essentielle pour développer le projet et ajouter de nouvelles fonctionnalités complexes de manière efficace. L'utilisation de SASS m'a appris à créer une esthétique moderne et à utiliser des animations CSS pour rendre le site plus attractif.
